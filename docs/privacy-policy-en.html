Corgi Design - AI-Powered Web Design Assistant

Last Updated: February 5, 2026

Thank you for using Corgi Design (hereinafter referred to as "the Extension"). We take your privacy very seriously. This Privacy Policy is intended to explain how we collect, use, and protect your information.

1. Information Collection

The Extension may process the following types of information during operation:

  • Web Page Content Data: When you actively use the Extension to analyze web design, we read the HTML structure, CSS styles, and page layout information of the current page to generate design suggestions.
  • Local Storage Data: Your settings preferences and configuration options are stored in the browser's local storage.

This Extension does NOT collect the following information:

  • Personal identification information (such as name, email, phone number)
  • Login credentials or passwords
  • Browsing history
  • Payment or financial information
  • Any sensitive personal data

2. Use of Information

The information we process is used only for the following purposes:

  • Analyzing the design structure of the current web page to provide you with intelligent design suggestions
  • Generating optimization plans and improvement suggestions
  • Saving your local settings preferences

3. Data Storage and Transmission

  • Local Storage: Your settings and preferences are stored only in your local browser and are not uploaded to any server.
  • AI Analysis: When using the AI analysis feature, web page structure information may be sent to the AI service provider for processing. We only send necessary page structure data and do not include any personal information.

4. Third-Party Services

This Extension may use third-party AI services to provide design analysis features. These services have their own independent privacy policies, and we recommend that you review the privacy terms of the relevant services.

5. Permissions Explanation

This Extension requests the following browser permissions:

  • storage: Used to save your local settings and preferences
  • activeTab: Used to access the content of the tab you are currently viewing for design analysis
  • scripting: Used to inject analysis scripts into pages to extract page design information
  • host_permissions (all URLs): Allows the Extension to run on any website so you can analyze the design of any web page

6. Data Security

We take reasonable technical measures to protect the security of your data. All local data is stored in your browser and is protected by browser security mechanisms.

7. Children's Privacy

This Extension is not directed at children under the age of 13. We do not knowingly collect personal information from children.

8. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page and the "Last Updated" date will be updated. We recommend that you review this policy regularly to stay informed of the latest information.

9. Your Rights

You have the right to:

  • Clear the local data stored by this Extension at any time through browser settings
  • Uninstall this Extension at any time to stop all data processing
  • Contact us for detailed information about data processing
